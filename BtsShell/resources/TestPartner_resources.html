[Documentation] This keyword is used to do BTS Commissioning in Site Manager\n
\n
Input:\n
\ | Input Parameters \ | Mandatory | Description |\n
\ | ${input_list} \ | Yes \ | a list including necessary info |\n
\n
\n
Usage:\n
| TestPartner SM BTS Commissioning | ${input_list} |\n
\n
\n
Note:\n
1) The input list should include the parameter BTS_page which specify the page to be updated, such as TRS_page=BTS_Settings\n
2) If you want to update multiple pages, you need to indicate it in BTS_page as follows\n
\ \ \ BTS_page=LTE_Carriers:BTS_Settings:Cell_Resources\n
3) If there's BTS_Settings in BTS_page, you need to input the field and value via the following parameters\n
\ \ \ BTS_Settings_field=BTS_name\n
\ \ \ BTS_Settings_value=BTS66\n
Similarly, if there's LTE_Carriers in BTS_page, you need to input the field and value via the following parameters\n
\ \ LTE_Carriers_field=Bandwidth\n
\ \ LTE_Carriers_value=20 MHz\n
Same for other pages.\n
4) In the same page, if you want to update multiple fields, conjoin them with ":".\n
5) On Cell_Resources page, Cell_Resources_field specify the cells to be updated. \n
\ \ \ \ Cell_Resources_field=Localcell1\n
\ \ \ Cell_Resources_value specify the field/value to be upated for the corresponding cell\n
\ \ \ \ \ Cell_Resources_field=field1=value1,field2=value2,field3=values\n
\ \ \ \ \ For multiple fields in a cell, conjoin them with ",".\n
\ \ \ \ \ For multiple cells, conjoin them with ":" just as the common field.\n
6) Not all fields for BTS commissioning have been added in this script.\n
\n
\n
Here's some examples for the input list\n
1) BTS Settings\n
1.1) Update BTS_name\n
\ \ \ BTS_page=BTS_Settings\n
\ \ \ BTS_Settings_field=BTS_name\n
\ \ \ BTS_Settings_value=BTS66\n
\n
2) Cell Resources\n
2.1) Update MIMO type\n
a) For single cell\n
\ \ \ BTS_page=Cell_Resources\n
\ \ \ Cell_Resources_field=Localcell1\n
\ \ \ Cell_Resources_value=MIMO_type=TXDiv\n
b) For 3 cell\n
\ \ \ BTS_page=Cell_Resources\n
\ \ \ Cell_Resources_field=Localcell1:Localcell2:Localcell3\n
\ \ \ Cell_Resources_value=MIMO_type=TXDiv:MIMO_type=Static Open Loop MIMO:MIMO_type=Closed Loop Mimo\n
c) For any cells\n
\ \ \ \ BTS_page=Cell_Resources\n
\ \ \ Cell_Resources_field=Localcell2:Localcell1\n
\ \ \ Cell_Resources_value=MIMO_type=TXDiv:MIMO_type=Static Open Loop MIMO\n
d) for multiple fields in multiple cells\n
\ \ \ \ BTS_page=Cell_Resources\n
\ \ \ Cell_Resources_field=Localcell1:Localcell2\n
\ \ \ Cell_Resources_value=MIMO_type=TXDiv,Power=5:MIMO_type=Static Open Loop MIMO,Power=5\n
2.2) Support the following fields as well\n
Power\n
\n
3) LTE Carriers\n
3.1) \ Update Bandwidth and EARFCN\n
\ \ BTS_page=LTE_Carriers\n
\ \ LTE_Carriers_field=Bandwidth:EARFCN\n
\ \ LTE_Carriers_value=10 MHz:1-38000,2-38000,3-38000\n
\n
\n
4) Update all the above fields\n
\ \ BTS_page=LTE_Carriers:BTS_Settings:Cell_Resources\n
\ \ BTS_Settings_field=BTS_name\n
\ \ BTS_Settings_value=BTS66\n
\ \ LTE_Carriers_field=Bandwidth:EARFCN\n
\ \ LTE_Carriers_value=10 MHz:1-38000,2-38000,3-38000\n
\ \ Cell_Resources_field=Localcell1\n
\ \ Cell_Resources_value=MIMO_type=TXDiv\n
\n
\n
5) Module Settings\n
5.1) Update Climate control profilling table. Click 'In use' and update Profile for FZNI 1.1.1;Update Profile for FZNI 1.3.1\n
BTS_page=Module_Settings\n
Module_Settings_field=Climate control profilling\n
Module_Settings_value=FZNI 1.1.1-1-Linear&FZNI 1.3.1-N-Linear\n
Note: 1) For multiple rows, conjoin with "&"\n
\ \ \ \ \ \ \ 2) N means ignore the responding column\n
\ \ \ \ \ \ \ 3) Colomn format\n
\ \ \ \ \ \ \ \ \ \ \ \ Module-In use-Profile
