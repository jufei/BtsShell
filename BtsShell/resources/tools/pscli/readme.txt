This tool is used to control the R&D parameter's value and Log level.
1. Please copy "ERadSwDomainLteMac.h" and "PsFeatureLevels.hpp" into the folder which contain "pscli.exe".
2. Confirm "pscli.exe" can reach the MCU's IP (192.168.255.1).
3. Use "pscli.exe" to change the R&D parameter or log level.

Note, more detail for pscli.exe can be checked by "rdtool.exe -h".
